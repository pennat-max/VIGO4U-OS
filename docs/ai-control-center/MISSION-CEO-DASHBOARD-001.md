## Mission

Build the CEO Dashboard first.

Dashboard นี้เป็น home screen ของ VIGO4U OS ERP

## Requirements

- Mobile-first Apps Script Web App
- Thai dashboard UI
- Mock data first
- Dark mode
- Card layout
- Large buttons

1. แดชบอร์ดผู้บริหาร
2. สถานะ VIGO4U ERP
3. สถานะ AI
4. สรุปธุรกิจ
5. สรุปการเงิน
6. สิ่งที่ติดอยู่
7. งานวันนี้
8. ปุ่มคำสั่งด่วน

## Goal

Dashboard must open and run.

## Implementation Target

- Folder: `VIGO4U-OS-CEO-Dashboard/`
- Web App: Google Apps Script
- Data: mock data from `getDashboardData()`

## Definition of Done

- Dashboard UI is Thai-first
- Dashboard is mobile-first
- Required sections exist
- Web App URL opens with HTTP 200
- Changes committed
- Changes pushed to GitHub
- Status docs updated
