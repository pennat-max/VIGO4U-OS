# AI Factory Workflow

GitHub is the source of truth for VIGO4U OS.

## Roles

ChatGPT:
- Product direction
- Architecture review
- Mission planning
- QA decisions

Codex:
- Code implementation
- Tests
- Deployment steps
- Status updates

Manus:
- Product review
- UX review
- QA review
- Security and ERP recommendations

## Working Rule
All work must end in GitHub, not only in chat or local files.

Before a task is considered done:
- Update status
- Record blockers
- Record next action
- Push changed files

## Stop Rule
Stop only when human approval or external account access is required.
